Oracle HCM Complete Tables Reference

This document serves as a comprehensive reference for Oracle HCM tables and their descriptions.

Table Name Description
PER_ALL_PEOPLE_F Primary table for employee information.
PER_JOB_HISTORY Stores job histories for employees.
PER_ORG_STRUCTURE_F Organizational structure and hierarchy.
PAY_PAYROLLS_F Payroll information for all employees.
PAY_CHECKS_F Stores check information for employee payments.
HR_ALL_ORG_UNITS All organizations within the system.
