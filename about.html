About AMZA

AMZA (amza.in) is a fast and practical reference platform designed for professionals working with Oracle Fusion Applications.

It provides quick access to commonly used HCM, SCM, and Finance tables, helping developers and consultants avoid time-consuming searches in complex documentation.

🎯 Purpose

  • Reduce time spent searching for table references
  • Provide quick technical and functional lookup
  • Act as a centralized Oracle Fusion reference workspace

⚙️ What You’ll Find

  • HCM Tables – Employee, Assignment, Payroll
  • SCM Tables – Procurement, Inventory (coming soon)
  • Finance Tables – GL, AP, AR (coming soon)

These references are useful for:

  • BI Publisher (BIP) Reports
  • OTBI Reports
  • HDL / FBDI Data Loads
  • SQL Query Development

🚀 Why AMZA?

  • Clean and simple interface
  • Fast access to important tables
  • Built for real-world Oracle Fusion use
